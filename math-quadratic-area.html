数学压轴题

二次函数与几何综合：面积最值问题

数学题解 2025-04-25
较难

题目

如图，抛物线 y = -x2 + bx + c 与 x 轴交于 A(-1, 0)、B(3, 0) 两点，与 y 轴交于点 C。

（1）求抛物线的解析式；

（2）点 P 是抛物线上的动点（不与 A、B 重合），连接 PA、PB，当三角形 PAB 的面积最大时，求点 P 的坐标及最大面积；

（3）在（2）的条件下，抛物线上是否存在点 Q，使得三角形 QAB 的面积等于三角形 PAB 面积的一半？若存在，求出所有符合条件的点 Q 的坐标。

解题思路

这道题是典型的二次函数与面积最值综合题。核心思路是：

  1. 待定系数法求抛物线解析式（第一问）
  2. 建立函数关系式表示面积，转化为二次函数最值问题（第二问）
  3. 方程思想：令面积等于某个值，解方程求点坐标（第三问）

详细解答

第一问：求抛物线解析式

因为 A(-1, 0)、B(3, 0) 在抛物线上，设抛物线解析式为交点式：

y = a(x + 1)(x - 3)

其中 a = -1（由题目给出的 y = -x2 + bx + c 可知二次项系数为 -1）。

展开验证：

y = -(x + 1)(x - 3) = -(x2 - 2x - 3) = -x2 + 2x + 3

答：抛物线的解析式为 y = -x2 + 2x + 3。

第二问：面积最值

关键思路：三角形 PAB 的底边 AB 是固定的，只需要让高最大即可。

AB 的长度 = 3 - (-1) = 4。

设 P(x, -x2 + 2x + 3)，则 P 到 AB（x 轴）的距离为 P 点的纵坐标的绝对值：

h = |-x2 + 2x + 3| = -x2 + 2x + 3（在 x 轴上方时为正）

三角形的面积：

S = (1/2) x AB x h = (1/2) x 4 x (-x2 + 2x + 3) = -2x2 + 4x + 6

这是一个关于 x 的二次函数，开口向下，有最大值。

当 x = -4 / (2 x (-2)) = 1 时，S 取最大值：

Smax = -2(1)2 + 4(1) + 6 = -2 + 4 + 6 = 8

将 x = 1 代入抛物线解析式求 P 点纵坐标：

y = -(1)2 + 2(1) + 3 = -1 + 2 + 3 = 4

答：当点 P 的坐标为 (1, 4) 时，三角形 PAB 的面积最大，最大面积为 8。

第三问：存在性问题

设三角形 QAB 的面积 = Smax / 2 = 4。

(1/2) x 4 x |-x2 + 2x + 3| = 4

|-x2 + 2x + 3| = 2

分两种情况：

情况一：-x2 + 2x + 3 = 2

-x2 + 2x + 1 = 0

x2 - 2x - 1 = 0

x = (2 +/- sqrt(4 + 4)) / 2 = 1 +/- sqrt(2)

x1 = 1 + sqrt(2) 时，y = 2；x2 = 1 - sqrt(2) 时，y = 2。

情况二：-x2 + 2x + 3 = -2

-x2 + 2x + 5 = 0

x2 - 2x - 5 = 0

判别式 Delta = 4 + 20 = 24 > 0

x = (2 +/- sqrt(24)) / 2 = 1 +/- sqrt(6)

x3 = 1 + sqrt(6) 时，y = -2；x4 = 1 - sqrt(6) 时，y = -2。

验证这些点不与 A、B 重合：-1 和 3 都不等于 1 +/- sqrt(2) 和 1 +/- sqrt(6)，满足条件。

答：存在四个符合条件的点 Q，坐标分别为 (1+sqrt(2), 2)、(1-sqrt(2), 2)、(1+sqrt(6), -2)、(1-sqrt(6), -2)。

方法总结

二次函数面积最值问题通用方法

  1. 确定不变量：找到固定的边（底边）
  2. 表示变量：用含 x 的式子表示高（或另一条边）
  3. 建立面积函数：S = (1/2) x 底 x 高，得到 S 关于 x 的二次函数
  4. 求最值：用配方法或顶点公式求最大/最小值
  5. 检验：检查最值点是否在允许范围内

举一反三

试着用同样的思路解决这道变式题：

抛物线 y = x2 - 4x + 3 与 x 轴交于 A、B 两点，点 P 是抛物线对称轴上的一个动点。当三角形 PAB 的周长最小时，求点 P 的坐标。

提示：这道题用到的不是面积最值，而是将军饮马模型——在对称轴上找一点 P，使 PA + PB 最小。利用抛物线的对称性，将 B 关于对称轴的对称点找到，再与 A 连线即可。
